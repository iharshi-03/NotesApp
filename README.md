# Simple Notes App - Jetpack Compose

## Objective
This is a single-screen Android application built to demonstrate modern Android development fundamentals, specifically focusing on Kotlin and the Jetpack Compose UI toolkit. 

## Features
* **View Notes:** Displays a list of existing notes in a clean, scrollable column (LazyColumn).
* **Add Notes:** Features a Floating Action Button that triggers a custom dialog for users to input a title and body for a new note.
* **State Management:** Uses `ViewModel` and `StateFlow` to manage the UI state reactively.
* **Modern UI:** Built entirely with Jetpack Compose using Material Design 3 components.

## Tech Stack
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Architecture:** MVVM (Model-View-ViewModel)
